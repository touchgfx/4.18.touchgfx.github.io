UI 개발 소개

기능성 UI 개발은 성공적인 임베디드 그래픽 제품을 만드는 데 매우 중요한 역할을 하므로, TouchGFX는 빠른 성능과 더불어 원활한 개발 경험을 제공하는 데 목표를 두고 있습니다.

UI 개발 섹션에서는 TouchGFX 애플리케이션 개발 방식에 대해 속속들이 살펴봅니다.

  • 소프트웨어 아키텍처 - TouchGFX 애플리케이션의 전반적인 아키텍처와 설계를 비롯해 TouchGFX Designer의 생성 코드와 사용자 코드 간 관계에 대해 설명합니다.
  • TouchGFX 작업 - TouchGFX 애플리케이션의 개발 워크플로에 대한 정보, 그리고 PC 시뮬레이터에서부터 그 밖에 다양하게 지원되는 IDE에 이르기까지 개발 프로세스에 사용되는 갖가지 도구에 대한 정보를 제공합니다.
  • 설계자 사용자 가이드 - TouchGFX Designer의 갖가지 구성요소를 사용하는 방법과 관련하여 폭넓은 지침과 팁, 그리고 유용한 기법을 제공합니다.
  • TouchGFX 엔진 기능 - 비트맵 캐싱, 부분 프레임버퍼, 다수의 언어 지원 등 TouchGFX 엔진의 갖가지 기능에 대한 정보를 제공합니다.
  • UI 구성요소 - 위젯에서부터 컨테이너에 이르기까지 TouchGFX에서 찾아볼 수 있는 모든 UI 구성요소에 대한 정보를 제공합니다.
  • 시나리오 - 개발자에게 일어날 수 있는 여러 가지 시나리오와 해결 방안을 소개합니다.
