UI開發簡介

開發一個功能性UI是擁有一個成功的嵌入式圖形產品的有機組成部分，因此，TouchGFX的目標不僅是提供快速性能，還提供流暢的開發體驗。

UI開發一章重點介紹TouchGFX應用程式的詳細開發流程：

  • 軟體架構 - 描述TouchGFX應用程式的總體架構和設計，以及從TouchGFX Designer生成的程式碼與用戶程式碼之間的關係。
  • 使用TouchGFX - 包含與TouchGFX應用程式的開發流程以及該過程中使用的不同工具有關的資訊 - 從PC模擬器到所支援的許多IDE。
  • 設計人員使用者指南 - 包含關於如何使用TouchGFX設計器的不同元件的廣泛指南、以及提示和技巧。
  • TouchGFX引擎特性 - 包含與不同TouchGFX引擎特性（如點陣圖緩存、部分影像緩衝區、多語言支援等）有關的資訊。
  • UI元件 - 包含與TouchGFX中每個UI元件（從小部件到容器）有關的資訊。
  • 場景 - 介紹開發人員可能遇到的不同場景，以及相應的解決方法。
